Features of Smart Guiding

An Android Application

5 Million+ Downloads

Smart Guiding is controlled from mobile devices with our Android application. The user can activate and use the application with only few touches. When the user taps the screen, the application connects to the device. Then the user can enable or disable the detection process with tapping the big voice button on the screen.

Portable Hardware

5 Million+ Downloads

Our hardware module mainly consist of an Ardunio Mega, ESP32 camera and CJMCU-531 VL53L1X laser sensor. The camera and the laser sensor gathers the necessary data from outside world and transfers it to user's mobile device via WI-FI.
