Features of Smart Guiding

An Android Application

5 Million+ Downloads

Smart Guiding is controlled from mobile devices with our Android application. The user can activate and use the application with only few touches. When the user taps the screen, the application connects to the device. Then the user can enable or disable the detection process with tapping the big voice button on the screen.

Portable Hardware

5 Million+ Downloads

Our hardware module mainly consist of an Ardunio Mega, ESP32 camera and CJMCU-531 VL53L1X laser sensor. The camera and the laser sensor gathers the necessary data from outside world and transfers it to user's mobile device via WI-FI.

Life Changing Capabilities

5 Million+ Downloads

Smart Guiding has designed for give warning about outside world to visually impaired people. When an object enters the field of view of the camera and the laser sensor, the mobile application gives warning with sound like â€œThere is a chair 30 cm in front of you". In every 10 second, it refreshes the information and give warning about the recent found object.

High Performance Object Detection Algorithm

5 Million+ Downloads

Smart Guiding uses a pre-trained YOLOv4 TFLite Object Detector for detecting the objects that the camera captures. It has capability of detecting different 80 object such as person, bicycle, car, motorbike, aeroplane, bus, train, truck, boat, etc. Credits to: Haroon Shakeel
